# Insert in Sorted Linked List

## Description
This project contains a C function that inserts a number into a sorted singly linked list while maintaining the sorted order.

## Function Prototype
```c
listint_t *insert_node(listint_t **head, int number);
```

## Return Value
- The address of the new node
- NULL if it failed

## Files
- `lists.h` - Header file containing function prototypes and struct definition
- `0-insert_number.c` - Implementation of the insert_node function
